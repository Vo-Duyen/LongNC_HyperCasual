# My Awesome Package

Mô tả package làm gì.

## Cài đặt

### Cách 1: Unity Package Manager
1. Mở Window → Package Manager
2. Click dấu + → Add package from git URL
3. Dán: `https://github.com/username/mypackage.git`

### Cách 2: Sửa manifest.json
Thêm dòng này vào file `Packages/manifest.json`:
```json
"com.yourname.mypackage": "https://github.com/username/mypackage.git"
